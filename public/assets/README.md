# ГЛУБИНА 400 — Ассеты

## Как заменить файл

Просто положи свой файл с нужным именем в нужную папку.
Движок берёт файлы по имени — ничего в коде менять не нужно.

## Видео-фоны (опционально)
**Папка:** `video/`
**Формат:** `.mp4`

Чтобы использовать видео как фон сцены, в JSON сцены укажи:
```json
{
  "background": "video/intro.mp4",
  "background_type": "video"
}
```

---

## Как используется в сценах (JSON)

```json
{
  "id": "prologue_1",
  "background": "corridor",
  "music": "ambient_main",
  "nodes": [
    {
      "character": "Лена",
      "expression": "scared",
      "text": "Алекс... ты жив."
    }
  ]
}
```

Движок автоматически ищет:
- Фон: `assets/images/bg/corridor.jpg`
- Спрайт: `assets/images/sprites/лена/scared.png` → `assets/images/sprites/lena/scared.png`
- Музыку: `assets/audio/bgm/ambient_main.mp3`
